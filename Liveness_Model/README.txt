LIVENESS MODEL 

liveneness_model.json - This file contains the model

liveness_model.ipynb - This Jupyter notebook contains the process of building the model

liveness_model.h5- Contains the weights for the liveness model

liveness.py - The python script that is run to bring up the webcam

imagetest_liveness.py - The python script that is run to check prediction for a single image

haarcascade_frontalface_default.xml - Face detection model
 
directions.txt - directions on how to use liveness.py and imagetest_liveness.py

requirements.txt - The requirements needed to run the model
